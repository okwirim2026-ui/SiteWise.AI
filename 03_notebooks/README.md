# Notebooks

Google Colab notebooks for the SiteWise.AI pipeline.

| Notebook | Purpose |
|----------|---------|
| 01_dataset_preparation_colab.ipynb | Image resizing, cleaning, train/val/test split |
| 02_yolo_training_colab.ipynb | YOLOv8 training on annotated HVAC dataset |
| 03_sam_segmentation_colab.ipynb | SAM segmentation refinement experiments |
| 04_inference_and_checklist_demo.ipynb | End-to-end pilot demo |

All notebooks are designed to run in Google Colab with GPU runtime enabled.
