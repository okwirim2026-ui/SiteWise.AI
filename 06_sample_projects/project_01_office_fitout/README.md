# Project 01 — Office fit-out HVAC duct inspection

## Description

Pilot demonstration project. Tests the SiteWise.AI workflow on office fit-out HVAC duct installation site images.

## Image sources

| Contributor | Images | Category |
|-------------|--------|----------|
| Shaima Husameldin | 53 | Duct components |
| Mohammed Musleh | 100 | Flexible ductwork |
| Peace Olaniyi | 49 | Wall penetrations, MEP openings |

## Folder contents

- dwg/ — original drawing files
- pdf/ — exported technical drawings
- site_images/ — site photos for AI inspection
- drawing_snapshots/ — cropped 2D drawing references
- ai_outputs/ — YOLO detections and SAM masks
- inspection_reports/ — generated checklist reports

## Inspection focus

- Duct count
- Support hanger presence
- Flexible duct geometry and condition
- Installation continuity
- Visible gaps or incomplete connections
- Insulation continuity features
